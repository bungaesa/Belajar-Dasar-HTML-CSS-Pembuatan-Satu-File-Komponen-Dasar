# Basic HTML & CSS Components

Latihan dasar HTML dan CSS dalam satu file untuk memahami struktur, elemen, selector, dan styling.  
Tugas ini cocok untuk pemula sebelum masuk ke Tailwind atau framework lain.

## Struktur File

Repository ini berisi 2 file utama:

index.html
styles.css

- index.html → berisi berbagai elemen HTML dalam satu halaman (header, list, table, card, form, dan lainnya)
- styles.css → berisi styling dasar menggunakan selector element, class, id, dan flexbox

## Tujuan Pembelajaran

- Memahami cara kerja tag HTML dasar
- Memahami perbedaan class dan id
- Bisa membuat struktur halaman sederhana
- Mengerti selector CSS (element, class, id)
- Memahami layout dasar menggunakan flexbox
- Mengerti spacing: margin & padding
- Membuat card, table, list, dan form sederhana

## Checklist Tugas

### HTML

- [ ] Membuat header dengan navigasi
- [ ] Membuat judul & paragraf (h1, h2, p)
- [ ] Menambahkan list (ul, li)
- [ ] Membuat tabel (table, thead, tbody)
- [ ] Membuat card section menggunakan div
- [ ] Membuat form dengan label, input, button
- [ ] Menambahkan footer
- [ ] Menggunakan id minimal 1 kali
- [ ] Menggunakan class minimal 3 kali

### CSS

- [ ] Styling header & nav
- [ ] Styling table
- [ ] Styling list
- [ ] Styling card
- [ ] Styling form
- [ ] Menggunakan element selector (p, h1)
- [ ] Menggunakan class selector (.card)
- [ ] Menggunakan id selector (#main-title)
- [ ] Menggunakan flexbox untuk layout
- [ ] Mengatur margin & padding
- [ ] Menggunakan warna (hex, rgb, atau rgba)
- [ ] Menggunakan border-radius & box-shadow
